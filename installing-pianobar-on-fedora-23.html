Installing Pianobar on Fedora 23

Pandora and pianobar are almost as important ‘tools’ on my development workflow as git and Emacs are. Not true, but sometimes I’m in the mood and capable of coding and listening to good music. Here is a quick tip on how to install pianobar on a Fedora system.

We start by cloning the repository:

git clone git@github.com:PromyLOPh/pianobar.git

Let’s install all the make dependencies at once, thank me later:

sudo dnf groupinstall -y "Development Tools" "Development Libraries"
sudo dnf install -y http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-stable.noarch.rpm http://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-stable.noarch.rpm
sudo dnf install -y libcurl libcurl-devel make automake gcc gcc-c++ kernel-devel ffmpeg ffmpeg-devel libgcrypt libgcrypt-devel libao libao-devel json-c json-c-devel

Next, build pianobar and install:

cd pianobar
gmake clean
gmake
sudo gmake install

Now, lets configure it:

mkdir ~/.config/pianobar
cp contrib/config-example ~/.config/pianobar/config

Go edit that file and add your Pandora username, password and set the autostart_station to your favorite personal station. To make pianobar’s output show in color uncomment the 3 format string lines. That’s it!

Put your headsets on, type pianobar and keep coding away!
