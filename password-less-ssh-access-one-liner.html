Password-less ssh access one-liner

So you are one of the guys that like to use a key pair to access your servers? Are you the one that ssh’s into the server, creates an authorized_keys file under the .ssh folder and places his public key in there?

Here is a nice short one-liner to do that:

ssh root@server "cat >> ~/.ssh/authorized_keys" < ~/.ssh/id_rsa.pub

This one-liner will redirect the local public key into the ssh command, which in turn will run the cat to the contents of the redirect and append that into the remote authorized_keys.

Bottom line, you end up with your local public key authorized in the remote server, thus a password-less access to it.

Enjoy, and keep hacking.
