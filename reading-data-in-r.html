Reading Data in R

One of the fundamental tasks a data scientist has to perform is loading the data to be analyzed.

On this recipe, I will demonstrate how to read different formats of data in R. To make this fully reproducible, we will be using some files I collected and uploaded to my site. Check the references section for credits.

We will show the following formats:

  • CSV
  • JSON
  • XML
  • Excel
  • Sqlite
  • HTML

Obtaining the data

# create a temp directory and get on it

Reading from sqlite database

library(RSQLite)
sqlite <- local_files[grepl("sqlite", local_files)]
conn <- dbConnect(drv=RSQLite::SQLite(), sqlite)
tables <- dbListTables(conn)
tables
dbGetQuery(conn,"SELECT name, summary FROM packages WHERE name like '%py%' LIMIT 5")
lapply(setdiff(tables, 'packages'), function(table) dbGetQuery(conn, paste("SELECT * FROM", table, "LIMIT 5")))
